# SCP 文件传输工具

## 工具特点

✅ **封装📦常用 scp 命令**

- 支持从服务器下载文件到本地
- 支持从本地上传文件到服务器
- 实时显示传输进度
- 最简单交互 - terminal 数字🔢交互
- 配置文件管理

✅ **跨平台支持** - 毕竟是 python 😁
- Windows
- macOS
- Linux

✅ **纯 Python 实现**
- 只使用 Python 标准库
- 无需安装额外依赖
- Python 3.6+ 即可运行

## 环境要求(目前只考虑 Windows)

- Python 3.6 或更高版本

  我是用 uv 做的统一管理, 所以有一些 uv 相关配置


- OpenSSH Client (scp) - Windows 10/11 内置
- PuTTY (pscp.exe)

## 使用方法

### 1. 运行脚本
```powershell
python scp_transfer.py
```

### 2. 选择操作
```
================================
    SCP 文件传输工具
================================

当前配置:
  服务器: username@ip:port
  本地路径: D:\xx\...
  服务器路径: /home/xx/...

请选择操作:
  1. 从服务器下载文件到本地
  2. 从本地上传文件到服务器
  3. 退出

请输入选项 (1-3):
```
